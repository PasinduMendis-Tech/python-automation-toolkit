# Python Automation Toolkit

A collection of beginner-friendly Python automation scripts built to solve real-world problems.

## Scripts Included

### 1. File Organizer
Automatically organizes files in a folder based on file type (Documents, Images, Music, etc.).

**Features:**
- Ignores macOS hidden system files
- Safe to run multiple times
- Prevents overwriting existing files

## Technologies Used
- Python 3
- OS file system handling
- Git & GitHub

## Purpose
This project demonstrates my ability to write practical Python automation scripts, handle edge cases, and follow clean version control practices.
